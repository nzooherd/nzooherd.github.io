我的2017

2018/02/15

2017就这样在外面的鞭炮声中结束了，细细品味这一年了，发现自己好像确实没干什么，或许我在逐渐的接受自己也是个平凡人的事实。

Java

在2017年上半年接触了Java语言，第一次接触面向对象语言，学得过程很轻松，Java课程的作业也做的很愉快，很享受这种编码的过程，每当手指在键盘上舞动跳跃，就仿佛在创造生命。这是在信安实验室感受不到的。在上半年的末尾还接触了tomcat，mysql等，学习JVM等，逐渐明确了自己的目标，Java研究工程师，工作后有机会转向人工智能。

Minc语言解释器

小学期和同学做的一个类C语言解释器，这也是我的第一个有真正的编码工作的项目。我负责了其中的语法处理等部分，做项目的过程中有痛快，但是写代码的过程却是很快乐的，尤其是当一个新的功能仅仅通过改了几行代码就完成的时候。

团队合作很重要。

基础

在新的学期，感觉课程内容终于进入了正轨，除了模电。

计算机网络，操作系统，数据库这些干货学习的过程还可以，烦人的则是复习的时候，本着六十分万岁的心态，总感觉多考一分又有何用，所以分数与我何加也，看到期末考试的成绩，随它去吧。学习不应该是为了分数。

复习

复习这条核心线贯穿了整个下半年。首先为了将精力放在找工作上，这学期选了四门专业课程加一门实验课，下学期是如此，如果一切正常的话，大四就没有课程了。

由于准备的是Java后台方向，在这学期的J2EE课程中可以说很拼了，一个多月的业余时间都用来搞这个项目，我希望写在简历中的项目可以做到问心无愧。在软件项目管理课上，放弃了自己小组的一个论坛项目，帮助另一个组做了一个聊天APP，从建库到编码可以说，我一个人完成了后台。还记得那两个星期，每天晚上都要去另一个宿舍找bug，心累。但是最后看到自己的后台成功将消息发出和接受还是很高兴的。

在10月初的时候制定了一系列的复习计划，整体来说是按照计划走的，但是仍然有一些任务没有完全完成，比如RBTree和AVL，现在还是没有记住。已经没有多少时间了。

其他

  • 学习了密码理论，看了一些比特币的知识，想实现一个自己的比特币，额，我还是太能想了。
  • 买了不少装备。
  • 驾照下来了。
