@nzooherd

2016年终总结

ipv6

一年下来，对于ipv6的一些研究，也许就是今年最大的收获了。现在想想，还清晰的记得突然之间就登录了谷歌账号的惊讶。随后大概一个月左右的时间真的有一种废寝忘食的感觉。

首先是懂得了一些DNS的基础知识，后来发现了xx-net,然后又认识了dd-wrt等一系列路由器固件，再最终走上自己搭建vps的道路，其中还了解到hexo这个博客框架，只能说一切都尽在一个缘字。如果不是当初，现在应该仍然还是一个用百度的小白，（现在是用狗哥的小白）。
以上的这些杂七杂八的东西，可以说没有哪个是软件学院的学生必须了解的，写到明年的德育答辩里，估计领导们也不知道你说的是啥，为什么我要去搞这些呢，e,　纯粹是兴趣吧，算了，一句话来激励一下。

追随自己的兴趣，成功将追随你。

算法和数据结构

对算法个数据结构的学习应该是这年的高收益，长半衰期的事件，准确应该说是下半年，上半学期荒废了不少时间。算法和数据结构是基础中的基础，不敢懈怠，不足的是数据结构课上的一些理论没能认真理解，倒是做了不少的题。大概是一个月前吧，看到学长对与应届生招聘的一些经验，决定刷一刷leetcode，现在刷了45道了，其中大概还荒废了了半个月的时间，相比那些两三周刷完leetcode的大神们，确实菜的抠脚啊。

天道酬勤

blog

下面就是博客了，完全是意料之外所得。当时搜索一些搭建vps的资料，无意中进入到学长的博客中，看到那个界面就心动了，真的有一种惊艳的感觉。于是便照着网上的教程DIY撸了一个，真特么的没技术含量。

python

python的学习应该也是这年比较重要的一个了。还记得当初想学点什么做网页的东西就不知道为什么糊涂的学了python，(捂脸.jpg。php和python的功能傻傻分不清）。
误打误撞下学完了python的基本语法，也写过一下最基础的爬虫脚本，登录脚本，最后还想着用python爆破一下老师的北理在线密码，只是一直没发成功报不了了之。python是一门非常灵活的语言。当初刷leetcode写一个求幂函数，用c写了几十遍也没过。python这样就过了。

class Solution(object):
  def myPow(self,x,n):
    import math
    return math.pow(x,n)

你特么在逗我。
不过不管怎么说，python都是一门非常优秀的语言，可以用来做许多事，恩，期望某一天可以用python写一个自己的网站。就这酱。

其他

  • 学习了html,css和基础的JavaScript。
  • 研究了一下树莓派，然并卵。有时间做个dd-wrt的路由器。
  • 加入了信安实验室。认识了一群大神。
  • 搞懂了一个网站通过域名被访问和建网站的基本流程。
  • 弄了台腾讯云，估计会陪伴大学四年。
  • vim撸起来就是爽。
  • 熟悉了一些Bash编程，Linux成为主要操作系统。

想想，应该就这些了，杂碎的不少，貌似更多的是对工具的研究。
希望新的一年里，可以成长的更多，玩一下github，做几个开源项目，甚至参加一次实习。Just do IT
